USDT.z - Stablecoin on Binance Smart Chain

USDT.z is a next-generation stablecoin pegged 1:1 to the US dollar, built on Binance Smart Chain (BEP-20). This site documents the reference contract, tokenomics, DeFi integrations, and audit artifacts for the project.

📦 Smart Contract Architecture

Core Components:

  • Context.sol — Base context for contract logic
  • IERC20.sol — Standard IERC20 interface
  • SafeMath.sol — Secure math
  • ERC20.sol — Standard token
  • TeamToken.sol — This project’s main contract

Admin Functions:

  • mint(address, uint256) - Mint new tokens (owner only)
  • burn(uint256) - Burn tokens (owner only)
  • pauseTrading() - Emergency stop
  • setFeePercent(uint256) - Set transfer fee
  • setBlacklist(address, bool) - Blacklist/unblacklist address
  • setWhitelist(address, bool) - Whitelist/unwhitelist address
  • setFeeWallet(address), updateOwner(address)
